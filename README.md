## commands

Install all dev tools
```shell
mise i
```

Install all deps if not done by `mise`
```shell
pn i
```

Update browserlist
```shell
pn update-browserslist-db latest
```

Generate types
```shell
pn types
```

Deploy to cloudflare
```shell
pn run deploy
```

Biome
```shell
pn biome check --write
```
